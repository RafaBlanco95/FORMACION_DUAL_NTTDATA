Proyecto del Taller Práctico 1 de JDBC

**Enunciado del Taller

-Ejecutar los scripts (SQL u Oracle según preferencias) facilitados por formador/a en un gestor de base de datos (PHPMyAdmin, SQL Developer, MySQL Workbench, HeidiSQL…) con conexión a base de datos establecida (SQL u Oracle).

-Crear un proyecto Apache Maven básico (arquetipo quickstart) al que añadir las dependencias necesarias  para establecer la conexión con la base de datos utilizado. 

-La clase App.java en su método main() debe contener una llamada a método privado que establezca la conexión con BBDD.

-Establecida la conexión se debe realizar una operación de consulta contra la base de datos para, posteriormente, mostrar la información retornada por pantalla.


**Resolución del Taller

Se ha creado un proyecto java en el que se han cumplido los requisitos del enunciado, aplicando las tecnologías:

-APACHE MAVEN
-LOGBACK
-JDBC 

Para la resolución del taller se ha utilizado el script SQL de ejemplo proporcionado por el formador para generar la base de datos, para después realizar la consulta nativa SQL en la conexión JDBC del proyecto.
